Overview

This is React based webpage for a Bakery. This webpage is list-based interface that allows users to add or remove various bakery items from the shopping cart and aggregates them to to provide a list of all the products added along with the total price.

More about the Application

This application consists of 15 different items which are displayed in list-based format and user can add these items to their shopping cart. There is a button which allows the user to add a specific item from the menu to the cart. The cart displayes item name, quantity and price along with the cart total. The quantity for each item is increased everytime the Add to Cart button is pressed.

Technical Details

The page is divided into three components: CartComponent, BakeryItem and BakeryCatalogue. BakeryItem component takes care of the structing of information of a variant of an iPhone and this is used by the BakeryCatalogue which also handles the functionality of the Add to Cart button. The CartComponent takes care of all the activites of the Cart such as displaying item name, quantity and price of the items in the Cart as well as aggregates and displayes the total cart value. I used props to pass any data to any of the components. State changes are only triggered whenever the user adds or updates the quantity of an item which is triggered whenever Add to Cart button is pressed.

Usability Principles Considered

- Design and layout is comprehensible at first sight.
- User has the ability to complete their task and does not get lost, they have all the information they need to know at any time, in which part of the application they are in, which actions can be executed and how they are triggered.
